# YFinance Tool Example

This example demonstrates how to use the **YFinanceTool** to build a financial analyst agent.

## Features
- Retrieve real-time stock prices
- Get daily highs, lows, and previous close data
- Calculate daily change percentage

## Usage

```bash
go run main.go
```

## How it works
The agent uses the `YFinanceTool` to query Yahoo Finance's API. It retrieves the latest market data for requested symbols and provides a professional analysis of the stock's performance.
